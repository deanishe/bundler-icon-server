Workflow Icon Generator

Generate icons for your Alfred 2 workflows. This webservice is a backend service for the Alfred bundler.

Usage

All implementations take the same arguments:

Argument Description Examples
icon The name of the icon/character. Use the preview links to view available icons and their names. adult, ban-circle, adjust, youtube
font The name of the font (see below). fontawesome, icomoon
colour A CSS colour without preceding #. May be 3 or 6 characters, upper- or lowercase. FFF, e3a2d1, 000000

Python bundler

If you're using bundler.py, call the icon() function with icon name, font name and CSS colour, e.g.: 

bundler.icon('align-center', 'fontawesome', '000000')
bundler.icon('arrow-circle-left', 'fontawesome', 'e5e5e5')

Miscellaneous bundler

If you're using the alfred.bundler.misc.sh bundler wrapper, you call it with icon name, font name and CSS colour, e.g.: 

alfred.bundler.misc.sh align-center fontawesome 000000
alfred.bundler.misc.sh arrow-circle-left fontawesome e5e5e5
